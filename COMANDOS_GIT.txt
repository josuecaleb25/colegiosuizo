=== COMANDOS PARA SUBIR TODO A GITHUB ===

1. Eliminar el repositorio git anidado de IEPeruanoSuizo:
   rd /s /q IEPeruanoSuizo\.git

2. Agregar todos los cambios:
   git add .

3. Hacer commit:
   git commit -m "Configurar backend para Render con PostgreSQL y WhiteNoise"

4. Subir a la rama josue:
   git push -f origin josue
